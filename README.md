Mountable Rails Engine for viewing paginated logs from a browser

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'logs'
```

And then execute:
```bash
$ bundle
```

Mount Engine:

`config/routes.rb`
```ruby
Rails.application.routes.draw do
  mount Logs::Engine => '/logs'
end
```

Look up on `/logs` route.
